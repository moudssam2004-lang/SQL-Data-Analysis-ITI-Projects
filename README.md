# SQL Data Analysis - ITI Database Projects

This repository contains complex SQL queries and data extraction tasks performed on the ITI (Information Technology Institute) database. It demonstrates hands-on experience with database querying, logic building, and data manipulation.

## 🛠️ Skills & Techniques Demonstrated:
- **Joins:** INNER JOIN, LEFT JOIN, and Self Joins for mapping complex relationships (e.g., matching students with their supervisors).
- **Window Functions:** Using `ROW_NUMBER()` and `PARTITION BY` to rank data within specific categories (e.g., finding the top 2 highest salaries per department).
- **Subqueries & CTEs:** Writing nested queries to perform multi-step filtering.
- **Aggregate Functions:** `MIN()`, `MAX()`, `AVG()`, `COUNT()` combined with `GROUP BY`.
- **Data Handling:** Managing NULL values using `COALESCE()` and `ISNULL()`.
- **Randomization:** Selecting random records using `NEWID()`.

## 📁 Files in this Repository:
- `MY ITI.sql`: Contains the complete SQL scripts and solutions for the business requirements.
